# Chiwawa Race 🏎️
My very first JS project. A browser betting game with Chiwawa racing.

<a href="http://chiwawarace.s3-website.eu-west-3.amazonaws.com" target="_blank">Live Demo</a>

## How to play
Chiwawa invites you to place a bet: pick the car you think will cross the finish line first. Click on it and hit **Start**. Four cars race to the finish — the winner is chosen at random. Guessed right — congratulations! If not — try again.

## Stack
Vanilla JS · HTML · CSS — no dependencies, single file.
